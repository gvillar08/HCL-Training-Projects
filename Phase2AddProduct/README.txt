For seeing how various features work, please reference the comments within the code
See pets.sql for the SQL I used

Steps for getting code to run:
	Run the provided pets.sql in MySQL Workbench
	Copy code into Eclipse in the same file structure as provided
	Edit hibernate.cfg.xml with your own SQL credentials as needed
	Build the Maven project
	Right click on the project > properties > deployment assembly > ensure that Maven Dependencies are present
	Run webapp.jsp on Tomcat server

Overall program notes:
	Once you select your first button, please give the program a minute to boot the hibernate utility. From then on, it should run smoothly without large waits
